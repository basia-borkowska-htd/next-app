# next-app

App created for purposes of learning Node.js and Next.js.

Tech Stack

- Next.js
- Node.js
- MongoDB
- Mantine
- Tailwind CSS

Run App Locally

`git clone https://github.com/basia-borkowska/next-app.git`

Server App
`cd next-app`

`npm install`

`npm start`

Client App
`cd client`

`npm install`

`npm run dev`
